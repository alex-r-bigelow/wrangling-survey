These categories are meant to be very broad; if none of them are a good fit for your dataset, please let us know with the "None of the above" option.

Tabular data typically consists of rows and columns; the intersection of a row and a column is called a cell. Tabular data can also include more complex datasets such as multiple linked / relational tables or tables containing multidimensional, complex, or nested cell structures.

Network data consists of nodes, connected with edges. This can include multivariate networks with node and/or edge attributes, as well as networks containing structures such as supernodes or hyperedges.

A hierarchy is a network with implied parent / child relationships. This can include linear, branching, flat, or overlapping hierarchies.

Spatial data may correspond to Fields include grids with any dimensionality, non-Euclidean coordinate systems, and fields with multidimensional attributes, such as vector or tensor fields. Geometry includes points in space, one-dimensional lines or curves, 2D surfaces or regions, or 3D volumes.
