A hierarchy is a network with implied parent / child relationships. This can include linear, branching, flat, or overlapping hierarchies.

Network data consists of nodes, connected with edges. This can include multivariate networks with node and/or edge attributes, as well as networks containing structures such as supernodes or hyperedges.

Tabular data typically consists of rows and columns; the intersection of a row and a column is often called a cell. Tabular data can also include more complex datasets such as multiple linked / relational tables or tables containing multidimensional, complex, or nested cell structures.
